💡 Smart Street Light Management System

📌 Overview
The Smart Street Light Management System is an IoT-based solution designed to optimize street lighting by automatically adjusting brightness based on ambient light conditions. It uses LDR sensors and microcontroller-based circuitry to ensure efficient energy usage and smart city integration.

 🎯 Objectives
- Automate street lighting based on environmental conditions  
- Reduce energy consumption  
- Improve efficiency of public lighting systems  
- Support smart city infrastructure  

 🧠 Working Principle
The system uses an LDR (Light Dependent Resistor) to detect ambient light intensity.  
Based on the detected light level:

- Daytime → Lights remain OFF  
- Low light → Lights turn ON  
- Night → Brightness increases automatically  

A microcontroller processes sensor data and adjusts light intensity using programmed control logic.

⚙️ Methodology
- LDR sensor continuously monitors light intensity  
- Microcontroller reads sensor values  
- Adaptive algorithm adjusts brightness levels  
- System responds dynamically to environmental changes
